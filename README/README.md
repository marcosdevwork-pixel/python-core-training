# Project 00 — Python Basics Training

**EN:**  
This project contains a series of essential Python exercises.  
Each exercise focuses on one fundamental concept needed for real-world automation.

**Exercises included:**
- Input & Output
- Variables and Types
- Conditions
- Loops
- Lists
- Dictionaries
- Functions
- File Handling

**Goal:**  
Build a strong foundation to create real and profitable Python automations.
