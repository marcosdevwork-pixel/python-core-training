# python-core-training

Serious Python training repository focused on core logic, problem-solving, and practical terminal-based projects.

# Python Training Projects

**EN:**  
This repository contains a collection of small yet serious Python training projects.  
Each project focuses on core logic, problem-solving, and practical skill-building for real-world programming.  
The goal is to develop strong fundamentals before moving on to automation, freelancing, and more advanced work.

**PT-BR:**  
Este repositório contém uma coleção de pequenos, porém sérios, projetos de treinamento em Python.  
Cada projeto foca em lógica central, resolução de problemas e construção de habilidades práticas para programação real.  
O objetivo é desenvolver fundamentos sólidos antes de avançar para automação, freelancing e trabalhos mais avançados.
